## Install
```bash
npm install neutrino-preset-egg-typescript --save
```

## Getting Started
```javascript
// package.json
"scripts": {
  "start": "neutrino start --use neutrino-preset-egg-typescript",
  "build": "neutrino build --use neutrino-preset-egg-typescript"
},
```
